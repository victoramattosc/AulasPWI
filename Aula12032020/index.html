Tags html

Tags Funcionalidade
(html) (/html) Envolve todo o documento html
(title) (/title) É o título do documento
(head) (/head) Usada normalmente para começar um html,é o cabeçalho ou de um jeito mais simples o nome da sua página
(body) (/body) É o corpo da página,basicamente é a onde escrevemos toda a página
(h1,h2,h3,h4,h5) (/h1,/h2,/h3,/h4,/h5) É usado para títulos e sub-títulos de uma página,quanto maior for o número menor o tamanho dos títulos e sub-títulos
(p) (/p) Usado para criar parágrafos
(ol) (/ol) É usado para criar uma lista ordenada por números
(ul) (/ul) É usado para criar uma lista sem uma ordem rígida ou seja não organizada númericamente
(li) (/li) É o item da lista
(< ! - -) (- - >) Usado para criar um comentário invisível para o telespectador que pode ser somente vista no código fonte
(style) (/style) É o estilo e forma da página por exemplo a cor ou o tamanho da fonte
(b) (/b) É o negrito na versão 4 do html
(strong) (/strong) É o negrito na versão 5 do html
(br /) É ultilizado para quebrar uma linha
(hr /) Faz uma linha na página
(i) (/i) Faz com que a fonte da frase seja em Itálico
